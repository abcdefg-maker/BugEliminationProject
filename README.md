# 波咕清除计划 (Bug Elimination Plan)

一款基于 Unity 开发的 2D 叙事冒险解谜游戏，讲述了一段发生在办公室中的多日剧情故事。

**开发团队：** P人一拍脑门

## 游戏简介

玩家将在一个办公室环境中，经历为期四天的故事线。通过与不同 NPC 对话、完成工作谜题、在危险中逃脱等玩法推动剧情发展。游戏采用标志位驱动的分支叙事，不同的游戏状态会触发不同的对话和事件。

## 核心玩法

- **对话系统** — 多角色、多阶段对话，支持角色立绘和语音，逐字显示效果
- **工作谜题** — 二进制序列选择与删除的益智小游戏
- **怪物追逐** — 第三天起触发的紧张逃脱环节，被抓到即死亡重来
- **昼夜切换** — 平滑的日夜过渡动画
- **场景状态持久化** — NPC 位置和物体状态在场景切换间保持一致

## 操作方式

| 按键 | 操作 |
|------|------|
| WASD / 方向键 | 移动 |
| 鼠标左键 | 推进对话 |
| R | 进入工作场景 |

## 游戏流程

1. **第一天** — 认识同事，与 Boss 和 Wang 对话，解锁工作场景
2. **第二天** — 剧情推进，NPC 状态变化
3. **第三天** — 逃脱谜题出现，怪物开始追逐
4. **第四天** — 最终对话与结局

## 技术栈

- **引擎：** Unity (2D)
- **语言：** C#
- **分辨率：** 1920×1080
- **渲染：** 2D 精灵渲染 + Y轴排序深度
- **UI：** Unity UI (uGUI) + TextMesh Pro
- **数据格式：** ScriptableObject（对话数据）、JSON（场景状态持久化）

## 构建与运行

### 直接运行
`exe/` 目录下包含已构建的 Windows 可执行文件 `波咕清除计划.exe`，可直接运行。

### 从源码构建
1. 使用 Unity Hub 打开本项目
2. 确保安装了对应版本的 Unity 编辑器
3. 在 Build Settings 中选择 PC, Mac & Linux Standalone 平台
4. 点击 Build 即可生成可执行文件
